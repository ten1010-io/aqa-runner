# cases/

Put your compiled **`cases.compiled.yaml`** in this folder, then launch the
runner (double-click `run.command` / `run.bat`, or run the CLI).

- The runner picks up `cases.compiled.yaml` here automatically — no path needed.
- If you keep several files here, name the one to run `cases.compiled.yaml`
  (or keep just one `.yaml`).
- If the folder has no YAML, the runner tells you to drop one in and opens this
  folder for you.

Compile a `cases.compiled.yaml` on an internet-connected machine with
`aqa-inspect` (see `ten1010-io/claude-toolkit`).
